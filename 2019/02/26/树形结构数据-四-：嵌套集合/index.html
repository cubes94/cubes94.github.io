简介

嵌套集合模型(Nested Set Model), 通过集合的包含关系表现分层结构, 父节点包含所有子节点区域。

也可以用树结构来表示，通过前序遍历赋值。

表结构

优缺点

优点：查询非常方便，同级节点可以根据比较左右值判断先后关系;
缺点：每次插入删除数据涉及到的更新内容太多;
